给Agent描述电路拓扑结构，让他据此写cadence 的 skill脚本 ，创建 schematic 有些困难，特别是遇到复杂的电路，skill 文档无法做到连线正确，以及若干问题。而且，生成的电路图与经验上有所区别，难以认出；
给 Agent 描述需求（电路结构），让其根据设计规则写 spice，并丢到 spectre 中仿真，根据仿真结果来分析，优化，修改电路，最后达到要求。关键在于形成一个闭环，直至最后的电路性能要求满足；
优化方法我打算用 LLM+BO 方法，具体脚本待写入；
整个流程：用户在 agent 聊天窗口输入需求-->agent 参考当前目录下的示例或知识库，根据需求及各种约束（包括 spice 脚本规范，工艺库相关的约束等）写 spice 脚本，spectre 仿真得到各项性能参数，性能参数与设定的目标对比，经过算法优化得到新的设计参数，再仿真，循环，直到达到步数上限或达到目标
从将得到的仿真结果与设定目标对比，到优化，到再仿真直至达到步数上限或达到目标，这些步骤可以通过自动化脚本实现（也就是写完 spice 脚本之后全部由自动化脚本实现）
